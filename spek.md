# 開発者用仕様書(仮)
##### 25G1133 山本龍之介
### 1.サーモンラン敵一覧　(スプラトゥーン３)
#### 機能
サーモンランの敵一覧とその特徴，倒し方を一覧表にし表示する
#### 構造
* id : 番号
* name : 名前
* kind : シャケの種類
* attack : 攻撃方法
* deth : 倒し方
* power: 別途討伐方法
### 2.D.LEAGUE 25-26Season 参画チーム一覧
#### 機能　
プロダンスリーグ**D.LEAGUE**における25-26Seasonに参画しているチームについての詳細を一覧表にする
#### 構造
* id : 番号
* name : チーム名
* jenre : チームメインジャンル
* people : 在籍人数
* leader : 現在のチームリーダー
* win : 優勝回数(レギュラーシーズン，CS両方)
### 3.なめこ人気ランキングTOP20 一覧
#### 機能
なめこの人気ランキングを一覧表にすることに加え，そのなめこの詳細と私個人の順位をつける
#### 構造
* id : 順位
* name : 名前
* kind : 特徴
* rank : レア度
* myranc : 私個人の順位
